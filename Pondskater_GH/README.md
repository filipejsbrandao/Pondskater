Pondskater Grasshopper plugin artifacts

This folder contains packaging and distribution artifacts for the Grasshopper plugin.

- `dist/` — packaged `.gha` files and platform zips.
- `prepare_dist.sh` — helper script to collect existing distribution artifacts into `dist/`.

Purpose: keep repository top-level focused on source (`Pondskater/`) while `Pondskater_GH/` holds plugin deliverables and packaging helpers.
